# 🚗 Car Service Database System

A Windows Forms based car service management system with database integration.

## 📋 Overview

This project is a desktop application developed in C# using Windows Forms.  
It allows a car service company to manage customers, vehicles, services and related data stored in a database.

The system supports data storage, modification and deletion through a user-friendly graphical interface.

---

## 🛠 Technologies Used

- C#
- .NET Framework / .NET Desktop
- Windows Forms (WinForms)
- SQL Database
- Visual Studio

---

## ✨ Features

- ➕ Add new service records
- ✏ Modify existing records
- ❌ Delete records
- 📄 Database integration
- 🖥 Multi-form application structure
- 🗂 Structured project solution
- 📦 Setup / Installer project included
